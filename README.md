# Employee Service

Employee Management Microservice built using:

- Spring Boot 3
- Spring Data JPA
- H2 Database
- OpenFeign
- Apache Kafka
- Docker
- JUnit 5
- Mockito

## Features

- Create Employee
- Validate Employee Data
- Call Location Service using Feign Client
- Publish Employee Events to Kafka
- Consume Events and Save Audit Records
- Dockerized Kafka Setup
